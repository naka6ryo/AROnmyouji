このフォルダにサウンドエフェクトを配置してください。

必要なファイル名:

- polygon_burst.mp3 -> 敵撃破時の「ポリゴンがはじける」効果音
- explosion.mp3 -> プレイヤー被ダメ時の爆発音
- attack_swipe.mp3 -> 攻撃（スイング）時のスワイプ音（強さに応じて再生速度を調整）

注意: リポジトリには `atttack.mp3` というファイルが存在します（typo）。
コードは既存のファイル名に合わせて `atttack.mp3` を参照するようになっています。
可能であればファイル名を `attack_swipe.mp3` にリネームしてください。

ブラウザで動作させる場合、ファイルは `game/assets/sfx/` に置いてください。
ファイル形式は `mp3` または `ogg`（環境に応じて）を推奨します。

注: 音声はユーザー操作（ボタンクリック等）の後にロード／再生すると自動再生制限の問題が起きにくいです。
